# Public evaluation artifacts

This directory contains the compact evaluation artifacts referenced by
[`EXPERIMENT_REPORT.md`](../EXPERIMENT_REPORT.md).

- `assets/`: representative PCA, t-SNE, Grad-CAM, task-attention, and
  task-token diagnostic figures.
- `metrics/`: compact CSV summaries for the four V2 models and the completed
  Task-aware diagnostics.

The original local result directories remain ignored because they contain
checkpoints, feature dumps, and many generated images. No UIEB dataset,
`.pt` checkpoint, or `.npz` feature dump is included here.
